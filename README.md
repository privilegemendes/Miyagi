# Miyagi 

A puzzle sliding game with a twist.
wax on wax off

## How to play

The goal of the game is to move the tiles around the board to match the goal state. The twist is that you can only move tiles that are adjacent to the empty space.

## Controls
| Key | Action |

| --- | --- |
    
| `w` | Move up |
| `a` | Move left |
| `s` | Move down |
| `d` | Move right |
| `r` | Reset |
| `q` | Quit |
